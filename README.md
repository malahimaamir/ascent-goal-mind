
# Ascent Goal Mind

Ascent Goal Mind is a productivity app designed to help you set, track, and achieve your goals. Cultivate a growth mindset and stay motivated with organized plans and progress insights.

## Features
- Set and organize short-term and long-term goals
- Visual progress tracking and reminders
- Motivational tools and resources
